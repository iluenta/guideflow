# Guía Rápida para tu TV Samsung UE43TU7125K

## Encendido y Navegación Básica

1.  **Encender/Apagar:** Presiona el botón de encendido (⏻) en el mando a distancia.

1.  **Botón Source/Input:** Busca un botón en el mando a distancia etiquetado como "Source" o "Input".
2.  **Selecciona la Fuente:** Presiona el botón y usa las flechas del mando para seleccionar la fuente deseada (HDMI1, HDMI2, TDT, etc.). Presiona OK para confirmar.

## Apps Preinstaladas

*   **Prime Video:** Si el mando tiene un botón directo de Prime Video, presiónalo para abrir la aplicación.

## Diagnóstico de Problemas

| Código/Señal | Significado | Solución |
|---|---|---|
| Sin imagen | TV no encendida | Verifica que el cable de corriente esté conectado y presiona el botón de encendido. |
| Sin sonido | Volumen muy bajo o silenciado | Aumenta el volumen o desactiva el silencio. |
| "Sin señal" en pantalla | Fuente incorrecta | Selecciona la fuente correcta (HDMI1, TDT, etc.) con el botón "Source". |
| Imagen borrosa | Mala señal de antena | Verifica la conexión de la antena o cable. |
| No funciona el mando | Baterías gastadas | Reemplaza las baterías del mando a distancia. |
| TV no responde | Posible bloqueo | Desconecta la TV de la corriente durante 1 minuto y vuelve a encenderla. |
| App no funciona | Problema de conexión a internet | Verifica la conexión WiFi. |
| Pantalla negra con sonido | Fallo interno | Apaga y enciende la TV. |

Si el problema persiste, contacta con el personal de soporte.

### Manual V1

# Manual de Uso del Microondas Infiniton

## Primeros Pasos

1.  Asegúrate de que el microondas esté conectado a una toma de corriente.
2.  Abre la puerta e introduce el alimento en un recipiente adecuado (ver sección "Recipientes").
3.  Cierra la puerta correctamente.

## Programas y Funciones

*   **Microwave:** Cocción por microondas.
    *   Pulsa el botón "Microwave".
    *   Usa el botón giratorio "Timer•Weight•Auto Menu" para ajustar el tiempo de cocción deseado.
    *   Pulsa el botón "Start/+30Sec./Confirm" para iniciar.
*   **Grill/Combi.:** Función grill o combinada (microondas + grill).
    *   Pulsa el botón "Grill/Combi.".
    *   Usa el botón giratorio "Timer•Weight•Auto Menu" para ajustar el tiempo de cocción deseado.
    *   Pulsa el botón "Start/+30Sec./Confirm" para iniciar.
*   **W.T./Time Defrost:** Descongelación por peso o tiempo.
    *   Pulsa el botón "W.T./Time Defrost".
    *   Usa el botón giratorio "Timer•Weight•Auto Menu" para seleccionar entre descongelación por peso (Weight) o por tiempo (Time).
    *   Ajusta el peso o el tiempo según corresponda con el botón giratorio.
    *   Pulsa el botón "Start/+30Sec./Confirm" para iniciar.
*   **Start/+30Sec./Confirm:** Inicia la cocción o añade 30 segundos al tiempo restante. También sirve para confirmar ajustes.
*   **Stop/Clear:** Detiene la cocción o borra la configuración actual.
*   **Auto Reheat:** Recalentamiento automático.
    *   Pulsa el botón "Auto Reheat".
    *   Usa el botón giratorio "Timer•Weight•Auto Menu" para seleccionar el tipo de alimento a recalentar.
    *   Pulsa el botón "Start/+30Sec./Confirm" para iniciar.
*   **Clock/Pre-Set:** Ajuste del reloj o preselección de programas.
    *   Para ajustar el reloj, mantén pulsado el botón "Clock/Pre-Set" durante unos segundos hasta que parpadeen los números.
    *   Usa el botón giratorio "Timer•Weight•Auto Menu" para ajustar la hora y los minutos.
    *   Pulsa el botón "Start/+30Sec./Confirm" para confirmar cada ajuste.
*   **Timer•Weight•Auto Menu:** Botón giratorio multifunción para ajustar el tiempo, el peso y seleccionar programas automáticos.

### Niveles de Potencia

Este microondas tiene varios niveles de potencia que se ajustan automáticamente según el programa seleccionado.

*   **Alto:** Para calentar líquidos y cocinar alimentos rápidamente.
*   **Medio:** Para cocinar alimentos más densos y recalentar platos.
*   **Bajo:** Para descongelar y mantener la comida caliente.

### Función Descongelación (W.T./Time Defrost)

1.  Pulsa el botón "W.T./Time Defrost".
2.  Gira el botón "Timer•Weight•Auto Menu" para elegir entre "Weight" (descongelar por peso) o "Time" (descongelar por tiempo).
3.  Si eliges "Weight", gira el botón para seleccionar el peso del alimento en gramos. Si eliges "Time", gira el botón para seleccionar el tiempo de descongelación en minutos.
4.  Pulsa el botón "Start/+30Sec./Confirm" para iniciar la descongelación. El microondas se detendrá automáticamente al finalizar.

### Recipientes

*   **Aptos:** Vidrio resistente al calor, cerámica, plástico apto para microondas.
*   **NO aptos:** Metal, aluminio, recipientes con adornos metálicos.

## Diagnóstico de Problemas

| Código/Señal | Significado | Solución |
|---|---|---|
| No enciende | No hay corriente | Comprueba que el microondas esté enchufado y que el enchufe funcione. |
| No calienta | Problema con el magnetrón | Contacta con el servicio técnico. |
| La puerta no cierra bien | Obstrucción o daño en la puerta | Limpia la puerta y verifica que no haya objetos que impidan el cierre. Si el problema persiste, contacta con el servicio técnico. |
| El plato giratorio no gira | Obstrucción o problema con el motor | Limpia el plato giratorio y la base. Asegúrate de que esté correctamente colocado. Si el problema persiste, contacta con el servicio técnico. |

## Consejos

*   Limpia el microondas regularmente con un paño húmedo y un detergente suave.
*   No uses productos abrasivos que puedan dañar la superficie.
*   Coloca una taza con agua y unas gotas de vinagre dentro del microondas y calienta durante unos minutos para eliminar olores.
*   Si no estás seguro de qué programa usar, empieza con un tiempo corto y ve añadiendo más según sea necesario.
*   Supervisa siempre la cocción, especialmente la primera vez que uses un programa nuevo.

# Guía Rápida para tu Microondas Infiniton

## Funcionamiento Básico

1.  **Microwave:** Pulsa para seleccionar la potencia del microondas.
    *   **100%:** Para calentar rápidamente.
    *   **80%:** Para cocinar alimentos.
    *   **50%:** Para cocción lenta o recalentar.
    *   **30%:** Para descongelar y mantener caliente.
    *   **10%:** Para mantener la comida caliente.
2.  **W.T./Time Defrost:** Para descongelar por tiempo. Introduce el tiempo con el selector giratorio y pulsa "Start/+30Sec./Confirm".
3.  **Start/+30Sec./Confirm:** Inicia el microondas o añade 30 segundos al tiempo de cocción. Confirma ajustes.
4.  **Stop/Clear:** Detiene el microondas o borra la configuración.
5.  **Auto Reheat:** Para recalentar alimentos automáticamente.
6.  **Selector Giratorio (Timer/Weight/Auto Menu):** Gira para ajustar el tiempo, el peso o seleccionar un menú automático.

## Descongelación por Tiempo

1.  Pulsa el botón **W.T./Time Defrost**.
2.  Gira el selector **Timer/Weight/Auto Menu** para ajustar el tiempo de descongelación deseado.
3.  Pulsa **Start/+30Sec./Confirm** para iniciar el proceso.

## Función Grill/Combi.

1.  Pulsa el botón **Grill/Combi.** para seleccionar la función grill o combinada.
2.  Ajusta el tiempo de cocción con el selector **Timer/Weight/Auto Menu**.
3.  Pulsa **Start/+30Sec./Confirm** para comenzar.

## Recipientes Aptos y No Aptos
